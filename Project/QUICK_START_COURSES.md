# Guia Rápido - Sistema de Cursos

## 🎮 Como Usar

### 1. Acessar Cursos
- Clique em "Cursos" no menu lateral (ou ícone de casa no mobile)
- Você verá todos os cursos disponíveis organizados por categoria

### 2. Filtrar por Categoria
- Use os botões no topo para filtrar:
  - Todos
  - Computação
  - Armazenamento
  - Banco de Dados
  - Redes

### 3. Selecionar um Curso
- Clique em qualquer card de curso
- Você verá todos os níveis do curso
- Níveis bloqueados (🔒) precisam que você complete o nível anterior

### 4. Jogar um Nível
- Clique em "Começar" no nível desbloqueado
- Responda as questões de múltipla escolha
- Veja o timer no topo (quanto mais rápido, mais XP!)
- Acerte questões seguidas para ativar o COMBO

### 5. Ver Resultado
Após completar o nível, você verá:
- ✅ Sua pontuação
- ⏱️ Tempo gasto
- ⭐ Acertos/Total
- ⚡ XP ganho
- 🏆 Bônus de tempo (Gold/Silver/Bronze)
- 🔥 Multiplicador de combo

### 6. Ganhar Medalhas
- Complete todos os níveis de um curso
- Ganhe uma medalha baseada no seu desempenho
- Medalhas aparecem no seu perfil

## 📊 Sistema de XP

### Como Ganhar XP
1. **Responder Questões**: 10-50 XP por questão
2. **Completar Níveis**: 100-400 XP bônus
3. **Bônus de Tempo**:
   - 🥇 Gold: +100 XP
   - 🥈 Silver: +50 XP
   - 🥉 Bronze: +25 XP
4. **Combo**: Multiplica XP (até x3)

### Subir de Nível
- Acumule XP para subir de nível
- Cada nível requer mais XP que o anterior
- Seu nível aparece no perfil

## 🏆 Conquistas

### Tipos de Conquistas
- **Especiais**: Primeira vitória, madrugador, coruja noturna
- **Velocidade**: Bônus de tempo, conclusão rápida
- **Precisão**: 100% de acerto, combos
- **Questões**: Responder 100, 1000 questões
- **Tempo**: 5h, 20h de estudo
- **Streak**: 7, 30 dias consecutivos

## 📱 Perfil

No seu perfil você pode ver:
- Nível atual e progresso
- Total de XP
- Cursos completados
- Medalhas conquistadas
- Conquistas desbloqueadas
- Streak atual
- Tempo total de estudo
- Taxa de acerto geral

## 🎯 Dicas para Maximizar XP

1. **Seja Rápido**: Tente ganhar bônus Gold em todos os níveis
2. **Mantenha o Combo**: Acerte seguido para multiplicar XP
3. **Estude Diariamente**: Mantenha seu streak ativo
4. **Complete Cursos**: Ganhe medalhas e XP bônus
5. **Desbloqueie Conquistas**: Cada conquista dá XP extra
